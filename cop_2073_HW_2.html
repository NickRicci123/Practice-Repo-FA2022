  1. (10 points) Define a function imperial_to_metric that takes in two arguments: a number of feet and a number of inches. The function should return the equivalent length in meters

To convert a length L from feet (ft) and inches (inches) to meters (m), one can use the relation: L(m) = L(ft) 0.3048 + L(inches) 0.0254

imperial_to_metric <- function(feet, inches) {
  
  meters <- feet * 0.3048 + inches * 0.0254
  
  return(meters)
}

imperial_to_metric(5,9)
